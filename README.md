## Description

This is API for project Mesto build on Node.js, Express.js, MongoDB and Mongoose.js.

## Features

– Update user info

– Create and delete card

– Add and remove like

– Validation middleware

– Error logging
 
  
## Run Locally

`npm run start` — run server  
`npm run dev` — run server with hot-reload
